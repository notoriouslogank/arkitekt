# Arkitekt

Arkitekt is a bash script for installing and configuring a new Linux
installation.

## Installation

Clone the repo.

```bash
git clone https://github.com/notoriouslogank/arkitekt.git
```

## Usage

```bash
cd src

sudo arkitekt.sh
```
